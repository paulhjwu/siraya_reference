# 西拉雅語詞綴 — 完整字母順序參考

所有詞綴均取自 Adelaar（2011），依詞綴形式的首字母排序。類型說明：**PRE**（前綴）· **SUF**（後綴）· **INF**（中綴/重疊）· **CIR**（周綴）· **CLI**（附著詞）

類別標記：**VOICE**（語態）· **VERB-CLASS**（動詞類別）· **TENSE**（時態）· **ASPECT**（體）· **MOOD**（語氣）· **CAUS**（使役）· **INCH**（起始體）· **CASE**（格）· **NUMBER**（數）· **NOML**（名詞化）· **ORIENT**（趨向）· **INV**（逆轉）· **TRANSF**（轉換）· **ADD**（添加）· **ADV**（轉折）

---

| 詞綴 | 類型 | 類別 | 語法功能 | 例子 |
|-------|------|----------|---------------------|---------|
| `a-` / `ä-` | PRE | **ORIENT** | 伴隨趨向（`com`）：「與……一起、帶著、共同」 | `a-para`「帶著走」；`a-kua`「服從」 |
| `aäw-` / `ääw-` | PRE | **INV** | 逆轉（`inv`）：「撤銷、反轉、取消-」 | `m-aäw-ka-kirix`「解開」；`m-aäw-dŭma`「轉身」 |
| `-a` | SUF | **MOOD** | 虛擬式（`sj`），施事導向（`ao`） | `l‹m›ikux-a`「願他回來」；`m-araraw-a`「將看見」 |
| `-al=` | SUF | **MOOD** | 虛擬式（`sj`）施事導向（`ao`）在附著詞 `=ato`/`=ăpa` 之前；`-a` 的異音語素 | `i-rua-al=ăpa-ko`「我也會來」；`ătaral-al=ato` |
| `-an` | SUF | **VOICE** · **NUMBER** · **NOML** | (1) 受事導向（`uo`）陳述式，處所/非核心論元（`loc`）凸顯；(2) 集合名詞（`coll`）後綴；(3) 動詞衍生處所名詞（`loc`） | (1) `ka-tälax-an`；(2) `kaäwlung-an`「人類」；(3) `ta-tulu-an`「火爐」 |
| `-aney` | SUF | **MOOD** · **VOICE** | 虛擬式＋受事導向（`sj.uo`），非核心/處所論元凸顯（`loc`） | `pää-tunun-aney`「將被交付」 |
| `-aneyl=` | SUF | **MOOD** · **VOICE** | `-aney` 在附著詞 `=ato`/`=ăpa` 之前的異音語素 | `pää-tunun-aneyl=ato` |
| `-aw` | SUF | **MOOD** · **VOICE** | 虛擬式＋受事導向（`sj.uo`），受詞彙限制 | `kan-aw`「讓它被吃」；`ătaral-aw` |
| `-awl=` | SUF | **MOOD** · **VOICE** | `-aw` 在附著詞 `=ato`/`=ăpa` 之前的異音語素 | `kan-awl=ato` |
| `-än` | SUF | **CASE** | 人名名詞上的斜格後綴（`obl`） | `tini-än`「他/她（斜格）」；`ĭmian-än`「我們（斜格，排他）」 |
| `Ca-` | INF | **ASPECT** · **NOML** | (1) 動詞的進行體（`rdp`）/靜態（`st`）；(2) 動詞衍生名詞（施事者、工具、受事者、抽象） | (1) `k‹m›a-kading`「（正在）看守」；(2) `da-dĭlux`「祭司」 |
| `Ca- … -an` | CIR | **NOML** · **ORIENT** | 動詞衍生處所名詞（`loc`）：「發生X的地方」（僅當 Ca- 與 -an 同時出現時取處所義） | `ta-tulu-an`「火爐/烤爐」；`vato-to-an`「多石之地」 |
| `-ey` | SUF | **MOOD** · **VOICE** | 虛擬式＋受事導向（`sj.uo`）；主要/默認形式 | `ătaral-ey`「願它被寬恕」；`kĭta-ey`「看它！」 |
| `-eyl=` | SUF | **MOOD** · **VOICE** | `-ey` 在附著詞 `=ato`/`=ăpa` 之前的異音語素 | `ătaral-eyl=ato` |
| `i-` / `ĭ-` | PRE | **ORIENT** | 位置趨向：「在空間或時間中（`loc`）」 | `m-i-rung`「坐」；`i-rua`「到達」 |
| `i- … -an` / `-ǝn` | CIR | **NOML** · **ORIENT** | 動詞衍生處所名詞（`loc`）：「X 發生的地方/時間/行為」 | `i-ka-kua-ǝn`「房間」；`i-da-rinux-an`「永恆、世界」 |
| `-i` | SUF | **MOOD** | 祈使式後綴（`imp`）（施事者或受事者） | `ad-i-ko`「把他們帶給我！」 |
| `-ing` | SUF | **VERB-CLASS** | 將疾病/身體部位名詞動詞化；烏特勒支手稿方言中 `-ǝn` 的變體 | `aĭngĭt-ing`「發燒」 |
| `ka-` | PRE | **VERB-CLASS** | (1) 第1類動詞標記（`v1`）（靜態/性質動詞，`ao1`）；(2) 出現於使役詞 `pa-ka-` 及名詞化周綴 `ka-…-an` 中 | (1) `ka-riang`「善良」；(2) `pa-ka-kuptix`「使純潔」 |
| `ka- … -an` | CIR | **NOML** · **NUMBER** | (1) 源自第1類動詞（`ao1`）的抽象/靜態名詞；(2) 名詞上的包括性總括（`incl`）：「所有的X、整個X」 | (1) `ka-patey-an`「死亡」；`ka-takut-an`「恐懼」；(2) `ka-tălax-an`「整個家戶」 |
| `ka- … -ǝn` | CIR | **NOML** | 源自第1類動詞（`ao1`）的抽象名詞：「X 的性質/狀態」 | `ka-muy-ǝn`「欲望」；`ka-ilx-ǝn`「力量」；`ka-kan-ǝn`「食物」 |
| `käwx-` | PRE | **INCH** | 起始體（`inch`）：「變成X」（第1類動詞，`ao1`，及名詞） | `käwx-darikax`「康復」；`käwx-vato`「變成石頭」 |
| `‹m›` | INF | **VOICE** · **VERB-CLASS** | 施事語態（`ao`），第3類動詞（`ao3`）；插入首輔音之後 | `k‹m›ĭta`「看見」；`d‹m›ingding`「審判」；`l‹m›ikux`「回來」 |
| `m-` | PRE | **VOICE** | 施事語態標記（`ao`）（大多數動詞類別） | `m-u-rbo`「進去」；`m-i-rung`「坐」 |
| `ma-` | PRE | **VOICE** · **VERB-CLASS** | 施事語態（`ao`），第1類（`ao1`，靜態）及第4類動詞（`ao4`） | `ma-patey`「死亡」；`ma-riang`「善良/偉大」 |
| `măka-` | PRE | **CASE** · **ORIENT** | 離格介詞/前綴：「從、自從、為了」 | `măka-rbo`「從裡面」；`măka-saat`「自某天起」 |
| `mi-` | PRE | **INCH** | 起始體（`inch`）：「變成X」（第1類動詞，`ao1`） | `mi-vulas`「變得悲傷」；`mi-variou`「替代」 |
| `-n` | SUF | **VOICE** | 受事導向（`uo`）陳述式；`-ǝn` 在末音節為元音的詞根後的異音語素 | `kĭta-n`「它被看見」；`ka-vana-n`「言語（受事導向）」 |
| `ni-` | PRE | **TENSE** | 過去時（`pst`） | `ni-k‹m›ĭta`「他看見了」；`ni-pa-u-pänäx`「被展示」 |
| `p-` | PRE | **CAUS** | 使役詞 `pa-` 在趨向前綴 `u-`/`äw-` 之前的縮略形式（`caus`） | `p-u-pänäx`「展示」；`p-u-rbo`「使進入」 |
| `pa-` | PRE | **CAUS** · **VERB-CLASS** | (1) 使役（`caus`）（一般）；(2) 第4類動詞構成前綴（`v4`） | (1) `pa-ka-patey`「殺死」；(2) `pa-i-sasu-an`「王國」 |
| `pa- … -an` | CIR | **NOML** · **VERB-CLASS** | 源自第4類動詞（`v4`）的動詞衍生名詞；處所或抽象義（`loc`） | `pa-kakĭtil-an`「婚禮」；`pa-titil-an`「意見分歧」 |
| `pa- … -ǝn` | CIR | **NOML** · **VERB-CLASS** | 源自第4類動詞（`v4`）的動詞衍生名詞；受事者/結果義 | `pa-ta-täutäux-ǝn`「教義」；`paku-su-ǝn`「誡命」 |
| `pa-ka-` | PRE | **CAUS** | 第1類動詞（`ao1`）的使役（`caus`） | `pa-ka-kuptix`「使純潔」；`pa-ka-riang-a`「拯救」 |
| `pa-pa-` | PRE | **CAUS** | 第4類動詞（`v4`）的使役（`caus`） | `pa-pa-reya`「締造和平」；`pa-pa-kulamux`「使穿衣」 |
| `paha-` | PRE | **TRANSF** | 轉換：「把X變成Y」 | `paha-pa-paul`「變成麵包」；`paha-viri-viri`「翻譯」 |
| `tama-` | PRE | **NOML** | 施事名詞化：「做V的人」 | `tama-dĭlux`「領導者」；`tama-ma-täutäux`「老師」；`tama-xail`「見證人」 |
| `-u` | SUF | **MOOD** | 祈使式後綴（`imp`） | `kading-u=kame`「救我們！」 |
| `u-` / `äw-` | PRE | **ORIENT** | 運動趨向（`mot`）：「朝向、進入」 | `m-u-rbo`「進去」；`m-äw-äwma`「去城市」 |
| `u- … -an` | CIR | **NOML** · **ORIENT** | 帶運動趨向的動詞衍生名詞（`mot`）；運動的結果/事件 | `u-va-va-tux-an`「慶典」；`u-va-var'x-an`「獎賞」 |
| `-ǝn` | SUF | **VOICE** · **NOML** · **VERB-CLASS** | (1) 受事導向（`uo`）陳述式（默認）；(2) 動詞衍生名詞（受事者/抽象）；(3) 將疾病名詞動詞化 | (1) `ătaral-ǝn`「被寬恕」；(2) `ka-muy-ǝn`「欲望」；(3) `dĭmi-dĭmi-ǝn`「癱瘓」 |
| `=ăpa` / `='pa` | CLI | **ADD** | 添加（`add`）：「也、還、又」；與 `ăsi` 連用 = 「甚至不／尚未」 | `i-rua-al=ăpa-ko`「我也會來」；`ăsi=ăpa`「甚至不」 |
| `=ato` / `='to` | CLI | **ASPECT** | 完成體（`prf`）：已完成或預期發生的事件 | `ma-takut=ato`「變得害怕」；`kidi=ato`「必須（發生）」 |
| `=da` / `=ra` | CLI | **ADV** | 轉折（`adv`）：「但是、然而、對比；引入新信息」 | `ma-lix=da`「（那）很強的」；`ausi=da`「但不是」 |

---

**類別說明**

| 標記 | 領域 | 含義 |
|--------|--------|---------|
| **VOICE** | 動詞性 | 施事導向與受事導向 |
| **VERB-CLASS** | 動詞性 | 第1、2、3或4類歸屬 |
| **TENSE** | 動詞性 | 過去時與非過去時 |
| **ASPECT** | 動詞性 | 完成體、進行體 |
| **MOOD** | 動詞性 | 虛擬式、祈使式 |
| **CAUS** | 動詞性 | 使役派生 |
| **INCH** | 動詞性 | 起始體「變成X」 |
| **CASE** | 名詞性 | 主格、斜格、離格 |
| **NUMBER** | 名詞性 | 集合、包括性總括 |
| **NOML** | 派生性 | 各類名詞化 |
| **ORIENT** | 派生性 | 運動、位置、伴隨趨向 |
| **INV** | 派生性 | 逆轉「撤銷/反轉」 |
| **TRANSF** | 派生性 | 轉換「把X變成Y」 |
| **ADD** | 話語性 | 添加「也、還」 |
| **ADV** | 話語性 | 轉折「但是、然而」 |

---

**備注**

1. 代詞附著詞（`=ko`、`=kaw`、`=kame`、`=kamu`、`=kĭta`、`=(m)au`、`=(m)hu` 等）不列於上表，因其構成封閉範式，已在代詞表中另行列出。
2. `măka-` 介於前綴與自由介詞之間；Adelaar 將其處理為通常與後接詞連寫。
3. 部分條目帶有兩個或三個類別標記（例如 `-an` 同時歸屬 **VOICE** + **NUMBER** + **NOML**）；這反映了語表形式的真實多功能性，而非分析上的歧義。
4. 異音語素（`-al=`、`-eyl=`、`-awl=`、`-aneyl=`）與其基礎形式共享類別，不在說明表中單獨列出。
