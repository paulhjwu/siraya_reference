# 西拉雅語代名詞系統

資料來源：Adelaar（2011），《西拉雅語：重建一門休眠的臺灣南島語言之音韻、語法與詞彙》，表2與表3。

西拉雅語代名詞分為四個系列——**獨立式**、**主格式**、**屬格式**與**斜格式**——涵蓋七個人稱／數範疇。第一人稱複數另外區分**包含式**（含聽者）與**排除式**（不含聽者）。

## 各系列說明

**獨立式**（Independent）為自由形式的強調代名詞，用於引入或強調所指人物。不依附於動詞，在格位上為中性。

**主格式**（Nominative）標記子句的句法主語。大多數為附著於動詞的後附詞（`=ko`、`=kaw`、`=kame` 等）；第三人稱採自由形式，由主格標記 `ta` 加獨立式代名詞構成（`ta teni`、`ta neini`）。

**屬格式**（Genitive）標記不在主語位置的施事者，或名詞的領有者。除第三人稱單數（`tin`）與第三人稱複數（`nein`）為自由形式外，其餘均為後附詞。部分人稱有兩個交替形式：`=(m)au`、`=(m)hu`、`=(m)ian`、`=(m)umi` 中的 `(m)` 為可選輔音，出現於特定語音環境。

**斜格式**（Oblique）涵蓋其他所有語義角色（受益者、方向、工具等）。均為自由形式，由獨立式代名詞加後綴 `-an` / `-än` 構成，例如 `ĭau` → `ĭau-an`、`teni` → `tini-än`。第三人稱單數斜格式 `tini-än` 在福音書文本中出現頻率最高。

## 備註

- 第一人稱複數**包含式**（`ĭmĭta` / `=kĭta`）指說話者與聽者雙方，即「我們（包括你）」。
- 第一人稱複數**排除式**（`ĭmian` / `=kame`）指說話者但不含聽者，即「我們（不包括你）」。
- 第三人稱的主格式與屬格式均為**自由形式**（非後附詞），與其他人稱不同。
- 屬格標記 `ki`（預設格標記）用於普通名詞，不用於代名詞。
- 縮寫說明：`1s` = 第一人稱單數 · `2s` = 第二人稱單數 · `3s` = 第三人稱單數 · `1pi` = 第一人稱複數包含式 · `1pe` = 第一人稱複數排除式 · `2p` = 第二人稱複數 · `3p` = 第三人稱複數
